# FLOWSTATE - WINDOWS SETUP INSTRUCTIONS

## Quick Fix for Your Current Issue

Your current `package.json` has a BOM (Byte Order Mark) encoding problem. Here's how to fix it:

### Option 1: Replace Everything (RECOMMENDED)

1. **Delete your current `C:\Flowstate\app` folder**

2. **Download the `flowstate-clean` folder from Claude**

3. **Rename it to `app`** and place it at `C:\Flowstate\app`

4. **Open PowerShell in that directory:**
   ```powershell
   cd C:\Flowstate\app
   ```

5. **Install dependencies:**
   ```powershell
   npm install
   ```

6. **Run the app:**
   ```powershell
   npm run dev
   ```

7. **Open browser:** http://localhost:3000

---

### Option 2: Just Replace package.json

1. **Delete your current** `C:\Flowstate\app\package.json`

2. **Copy the new** `package.json` **from** `flowstate-clean` **to** `C:\Flowstate\app\`

3. **In PowerShell:**
   ```powershell
   cd C:\Flowstate\app
   Remove-Item -Recurse -Force node_modules
   npm install
   npm run dev
   ```

## Troubleshooting

**If npm install is slow:**
```powershell
npm cache clean --force
npm install --legacy-peer-deps
```

**If port 3000 is busy:**
```powershell
npm run dev -- -p 3001
```
Then visit http://localhost:3001

**If you get module errors:**
```powershell
Remove-Item -Recurse -Force node_modules, package-lock.json
npm install
```
